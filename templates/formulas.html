Excel Formulas

Mathematical Formulas

Formula Name Short Description
SUM Sum Adds all the numbers in a range of cells.
AVERAGE Average Returns the average (arithmetic mean) of its arguments.
MIN Minimum Returns the smallest number in a set of values.
MAX Maximum Returns the largest number in a set of values.
PRODUCT Product Multiplies all the numbers given as arguments.
ROUND Round Rounds a number to a specified number of digits.
ROUNDUP Round Up Rounds a number up, away from zero.
ROUNDDOWN Round Down Rounds a number down, toward zero.
CEILING Ceiling Rounds a number up to the nearest multiple of significance.
FLOOR Floor Rounds a number down to the nearest multiple of significance.
MOD Modulus Returns the remainder after a number is divided by a divisor.
ABS Absolute Returns the absolute value of a number.
POWER Power Returns the result of a number raised to a power.
SQRT Square Root Returns the square root of a number.

Text Formulas

Formula Name Short Description
CONCATENATE Concatenate Joins two or more text strings into one string.
LEFT Left Returns the first character(s) in a text string based on the number specified.
RIGHT Right Returns the last character(s) in a text string based on the number specified.
MID Mid Returns a specific number of characters from a text string starting at the position specified.
LEN Length Returns the number of characters in a text string.
UPPER Upper Case Converts a text string to uppercase letters.
LOWER Lower Case Converts a text string to lowercase letters.
TRIM Trim Removes all spaces from a text string except for single spaces between words.
TEXT Text Formats a number and converts it to text.
SEARCH Search Finds one text string within another text string and returns the position of the starting character of the text.
FIND Find Returns the starting position of one text string within another text string, case-sensitive.
REPLACE Replace Replaces part of a text string with a different text string.
SUBSTITUTE Substitute Substitutes new text for old text in a text string.

Logical Formulas

Formula Name Short Description
IF If Returns one value if a condition is true and another value if it's false.
AND And Returns TRUE if all its arguments are TRUE.
OR Or Returns TRUE if any argument is TRUE.
NOT Not Reverses the logic of its argument.
IFERROR If Error Returns a value if an error is found; otherwise, returns the result of the formula.
IFNA If NA Returns a value you specify if a formula returns the #N/A error value; otherwise, returns the result of the formula.
SWITCH Switch Evaluates an expression against a list of values and returns the result corresponding to the first matching value.
CHOOSE Choose Returns a value from a list of values based on an index number.
ISBLANK Is Blank Returns TRUE if the specified cell is blank.
ISERROR Is Error Returns TRUE if the value refers to any error value.

Lookup and Reference Formulas

Formula Name Short Description
VLOOKUP Vertical Lookup Looks for a value in the first column of a table and returns a value in the same row from a column you specify.
HLOOKUP Horizontal Lookup Looks for a value in the top row of a table and returns a value in the same column from a row you specify.
INDEX Index Returns the value of an element in a table or array selected by the row and column number indexes.
MATCH Match Returns the relative position of an item in an array that matches a specified value.
CHOOSE Choose Returns a value from a list of values based on an index number.
OFFSET Offset Returns a reference to a range that is a specified number of rows and columns from a cell or range of cells.
TRANSPOSE Transpose Returns a horizontal range of cells as a vertical range and vice versa.
LOOKUP Lookup Looks in a one-row or one-column range (array) for a value and returns a value from the same position in a second one-row or one-column range.
COLUMN Column Returns the column number of a reference.
ROW Row Returns the row number of a reference.

Date and Time Formulas

Formula Name Short Description
TODAY Today Returns the current date.
NOW Now Returns the current date and time.
DATE Date Returns the number that represents the date in Microsoft Excel date-time code.
DAY Day Converts a serial number to a day of the month.
MONTH Month Converts a serial number to a month.
YEAR Year Converts a serial number to a year.
HOUR Hour Converts a serial number to an hour.
MINUTE Minute Converts a serial number to a minute.
SECOND Second Converts a serial number to a second.
WEEKDAY Weekday Converts a serial number to a day of the week.
EDATE Edate Returns the serial number of the date that is the indicated number of months before or after the start date.
NETWORKDAYS Network Days Returns the number of whole workdays between two dates.
EOMONTH End of Month Returns the serial number for the last day of the month that is the indicated number of months before or after the start date.
