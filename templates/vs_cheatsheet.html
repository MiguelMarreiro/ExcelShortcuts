Visual Studio Code Cheatsheet

General Shortcuts

Advice Name Short Description
Ctrl + P Quick Open Quickly open files by name
Ctrl + Shift + P Command Palette Access all commands in VS Code
Ctrl + / Toggle Line Comment Comment or uncomment the current line
Ctrl + B Toggle Sidebar Show or hide the sidebar
Ctrl + ` Toggle Terminal Show or hide the integrated terminal

Editing

Advice Name Short Description
Alt + Up/Down Move Line Move line up or down
Shift + Alt + Down Duplicate Line Duplicate the current line
Ctrl + D Add Selection to Next Find Match Select the next occurrence of the current selection
Ctrl + L Select Line Select the entire line
Ctrl + Shift + K Delete Line Delete the current line

Navigation

Advice Name Short Description
Ctrl + Shift + O Go to Symbol Quickly navigate to a symbol in the file
Ctrl + Shift + M Show Problems Show errors and warnings in the project
F8 Next Error or Warning Navigate to the next error or warning
Ctrl + Tab Switch Editor Cycle through open editors
Ctrl + G Go to Line Navigate to a specific line number

Extensions

Advice Name Short Description
Prettier Code Formatter Automatically format your code
ESLint Linting Find and fix problems in JavaScript
Live Server Local Development Server Launch a local development server with live reload
Bracket Pair Colorizer Bracket Highlighting Colorize matching brackets for easier readability
GitLens Git Integration Enhance Git capabilities directly in VS Code

Settings

Advice Name Short Description
editor.fontSize Font Size Set the font size in the editor
editor.tabSize Tab Size Set the number of spaces per tab
files.autoSave Auto Save Automatically save files after delay
editor.wordWrap Word Wrap Wrap lines exceeding the editor width
workbench.colorTheme Color Theme Change the color theme of the editor

Debugging

Advice Name Short Description
F9 Toggle Breakpoint Set or remove a breakpoint
F5 Start/Continue Start or continue debugging
Shift + F5 Stop Stop debugging
F11 Step Into Step into the function call
Shift + F11 Step Out Step out of the current function

Git

Advice Name Short Description
Ctrl + Shift + G Source Control Open the source control view
Ctrl + Enter Commit Commit changes
Ctrl + Shift + U View Output Show the output panel
Ctrl + Shift + H Replace Open search and replace
Alt + Shift + R Rename Symbol Rename all occurrences of a symbol

Emmet Abbreviations

Advice Name Short Description
div.container Container Div Creates a div with the class 'container'
ul>li*5 List Creates an unordered list with five list items
table>tr*3>td*2 Table Structure Creates a table with three rows and two columns
a[href="#"] Link Creates a hyperlink with a placeholder href
img[src alt] Image Tag Creates an image tag with src and alt attributes

Markdown

Advice Name Short Description
# Heading 1 Create a top-level heading
## Heading 2 Create a second-level heading
`code` Inline Code Display inline code snippets
``` Code Block Create a block of code
- List Create a bulleted list
