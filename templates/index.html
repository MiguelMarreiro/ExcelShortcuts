Excel Shortcuts

Editing Shortcuts

Shortcut Name Description
Ctrl + C Copy Copies the selected cells to the clipboard.
Ctrl + X Cut Removes the selected cells and copies them to the clipboard.
Ctrl + V Paste Inserts the contents of the clipboard into the selected cells.
Ctrl + Z Undo Reverses the last action.
Ctrl + Y Redo Reverses the last undo action.

Navigation Shortcuts

Shortcut Name Description
Arrow Keys Move Selection Moves the selection in the direction of the arrow key pressed.
Ctrl + Arrow Keys Jump to Edge Jumps to the edge of the data region in the direction of the arrow key.
Home Move to Start of Row Moves the selection to the beginning of the current row.
Ctrl + Home Move to Start of Worksheet Moves the selection to the top left corner of the worksheet (A1).
Ctrl + End Move to Last Cell Moves the selection to the last cell with content in the worksheet.

Formatting Shortcuts

Shortcut Name Description
Ctrl + B Bold Applies or removes bold formatting to the selected cells.
Ctrl + I Italic Applies or removes italic formatting to the selected cells.
Ctrl + U Underline Applies or removes underline formatting to the selected cells.
Alt + H, H Font Color Opens the font color menu to change the text color.
Alt + H, B Border Opens the borders menu to apply border styles to the selected cells.
