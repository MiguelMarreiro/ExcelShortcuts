Excel Shortcuts

General Shortcuts

Shortcut Name Short Description
Ctrl + N New Workbook Creates a new workbook.
Ctrl + O Open Workbook Opens an existing workbook.
Ctrl + S Save Workbook Saves the current workbook.
F12 Save As Opens the Save As dialog box.
Ctrl + W Close Workbook Closes the current workbook.
Ctrl + F Find Opens the Find dialog box.
Ctrl + H Replace Opens the Replace dialog box.
Alt + F4 Close Excel Closes Excel.
Ctrl + P Print Opens the Print dialog box.

Navigation Shortcuts

Shortcut Name Short Description
Ctrl + Arrow Key Move to Edge of Data Region Moves to the edge of the current data region in a worksheet.
Ctrl + Home Go to Start of Worksheet Moves to the beginning of the worksheet.
Ctrl + End Go to End of Worksheet Moves to the last cell with data in the worksheet.
Page Up Scroll Up Scrolls the screen up one page.
Page Down Scroll Down Scrolls the screen down one page.
Alt + Page Down Move One Screen Right Moves one screen to the right in the worksheet.
Alt + Page Up Move One Screen Left Moves one screen to the left in the worksheet.
Tab Next Cell Moves to the next cell to the right.
Shift + Tab Previous Cell Moves to the previous cell to the left.

Formatting Shortcuts

Shortcut Name Short Description
Ctrl + B Bold Applies or removes bold formatting.
Ctrl + I Italic Applies or removes italic formatting.
Ctrl + U Underline Applies or removes underline formatting.
Ctrl + 1 Format Cells Opens the Format Cells dialog box.
Alt + H, H Fill Color Opens the Fill Color menu.
Alt + H, A, C Center Align Center aligns the selected cells.
Alt + H, A, L Left Align Left aligns the selected cells.
Alt + H, A, R Right Align Right aligns the selected cells.
Ctrl + Shift + $ Currency Format Applies currency format with two decimal places.
Ctrl + Shift + % Percentage Format Applies percentage format with no decimal places.
Ctrl + Shift + # Date Format Applies date format.

Editing Shortcuts

Shortcut Name Short Description
Ctrl + C Copy Copies the selected cells.
Ctrl + X Cut Cuts the selected cells.
Ctrl + V Paste Pastes the copied or cut cells.
Ctrl + Z Undo Undoes the last action.
Ctrl + Y Redo Redoes the last undone action.
Delete Clear Clears the contents of the selected cells.
Ctrl + D Fill Down Copies the cell contents or formula from the top cell to the selected cells below.
Ctrl + R Fill Right Copies the cell contents or formula from the leftmost cell to the selected cells to the right.
Ctrl + - Delete Cells Opens the Delete dialog box to delete the selected cells.
Ctrl + Shift + + Insert Cells Opens the Insert dialog box to insert new cells.

Formula Shortcuts

Shortcut Name Short Description
Alt + = AutoSum Inserts the SUM function.
Ctrl + ` Show Formulas Displays all formulas in the worksheet.
Ctrl + Shift + Enter Array Formula Enters an array formula.
F4 Cycle Cell Reference Types Cycles through absolute and relative references.
Shift + F3 Insert Function Opens the Insert Function dialog box.
Ctrl + A Select All Selects all cells in the worksheet.
Ctrl + Shift + L Toggle Filter Adds or removes filters in the selected range.
Ctrl + Shift + U Toggle Formula Bar Expands or collapses the formula bar.
Ctrl + Alt + F9 Recalculate All Recalculates all formulas in all open workbooks.
Shift + F9 Recalculate Sheet Recalculates formulas in the active worksheet.
