Jinja Templates Overview

What is Jinja?

Jinja is a modern and designer-friendly templating language for Python, modeled after Django's templates.

Jinja Documentation

The official Jinja documentation can be found at: Jinja Documentation.

Topics Covered

  • Jinja Syntax
  • Control Structures
  • Filters
  • Macros

Jinja Syntax

In Jinja, you can use curly braces to output data or control structures.

{{ variable }}

Control Structures

Control structures like loops and conditionals can be used in Jinja templates.

{% if condition %}
    This will be displayed if the condition is true.
{% endif %}

{% for item in list %}
    {{ item }}
{% endfor %}

Filters

Filters modify values before they are displayed.

{{ variable|filter_name }}

Macros

Macros are like functions in Jinja and allow you to define reusable snippets of code.

{% macro render_item(item) %}
    
{{ item.name }}

{% endmacro %}

Useful Shortcuts

  • Output variable: {{ variable }}
  • Control with conditionals: {% if ... %} ... {% endif %}
  • Looping: {% for ... in ... %} ... {% endfor %}
  • Include another template: {% include 'template.html' %}
