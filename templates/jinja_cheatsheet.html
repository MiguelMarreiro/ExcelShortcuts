Jinja Template Core Formulas
Formula Name Description
{{ variable }} Variable Output Outputs the value of a variable
{{% for item in list %}} For Loop Loops through each item in a list
{{% if condition %}} Conditional Statement Executes code if the condition is true
{{% block block_name %}} Block Definition Defines a block that can be overridden
{{% extends 'base.html' %}} Template Inheritance Inherits from a parent template
{{% include 'file.html' %}} Include Template Includes another template
{{ variable | filter }} Filter Application Applies a filter to a variable
{{% set variable = value %}} Set Variable Sets a variable to a specified value
{{% macro function_name(args) %}} Macro Definition Defines a reusable chunk of template code
{{ variable|join(', ') }} Join Filter Joins a list into a string with specified separator
Jinja String Manipulation Formulas
Formula Name Description
{{ variable | length }} Length Filter Gets the length of a variable
{{ variable | upper }} Upper Filter Converts a string to uppercase
{{ variable | lower }} Lower Filter Converts a string to lowercase
{{ variable | capitalize }} Capitalize Filter Capitalizes the first letter of the string
{{ variable | title }} Title Filter Capitalizes the first letter of each word
{{ variable | trim }} Trim Filter Removes whitespace from the beginning and end
{{ variable | replace('old', 'new') }} Replace Filter Replaces occurrences of 'old' with 'new'
{{ variable | slice(start, end) }} Slice Filter Slices the string from start to end indices
{{ variable | striptags }} Strip Tags Filter Removes HTML tags from the string
{{ list | first }} First Filter Gets the first item in a list
Jinja Date & Time Formulas
Formula Name Description
{{ variable | datetimeformat('%Y-%m-%d') }} Date Format Filter Formats a datetime object to a string
{{ variable | now('%Y-%m-%d') }} Current Date Filter Gets the current date formatted as specified
{{ variable | timeuntil }} Time Until Filter Calculates time until a given date
{{ variable | timedelta(days=1) }} Time Delta Filter Adds a time delta to a date
{{ date | date('Y-m-d H:i') }} Date Filter Formats a date object to a specific string format
{{ variable | ago }} Time Ago Filter Formats as time ago from a given date
{{ variable | timestamp | date('*') }} Timestamp to Date Converts a timestamp to a date
Jinja Math Formulas
Formula Name Description
{{ value1 + value2 }} Addition Adds two values together
{{ value1 - value2 }} Subtraction Subtracts value2 from value1
{{ value1 * value2 }} Multiplication Multiplies two values
{{ value1 / value2 }} Division Divides value1 by value2
{{ value1 // value2 }} Floor Division Returns the largest integer less than or equal to the division
{{ value1 % value2 }} Modulus Returns the remainder of division
{{ value | round }} Round Filter Rounds a value to the nearest integer
{{ value | sum }} Sum Filter Calculates the sum of a list of numbers
{{ value | min }} Min Filter Finds the minimum value in a list
{{ value | max }} Max Filter Finds the maximum value in a list
Jinja Miscellaneous Formulas
Formula Name Description
{{ variable | default('default value') }} Default Filter Returns a default value if variable is undefined
{{ variable | length | default(0) }} Length with Default Returns length or 0 if variable is undefined
{{ variable | replace('old', 'new') }} Replace Filter Replaces occurrences of 'old' with 'new'
{{ variable | indent(4) }} Indent Filter Indents lines of text by a specified number of spaces
{{ variable | list }} List Filter Converts the variable to a list
{{ variable | sort }} Sort Filter Sorts a list
{{ variable | unique }} Unique Filter Removes duplicate items from a list
{{ variable | safe }} Safe Filter Marks a string as safe to render
{{ variable | tojson }} To JSON Filter Converts the variable to a JSON string
{{ variable | jsonify }} Jsonify Filter Converts the variable into a JSON object
